## Getting Started

```bash
# Clone the repository
git clone https://github.com/OWASP-STUDENT-CHAPTER/oss-programs.git
cd oss-programs

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
